# JSC370 Project: PM2.5 Prediction Analysis

This repository contains the project work for the JSC370 course, focusing on the analysis and prediction of PM2.5 levels using AQS and NOAA data.

## 🌐 Live Website
You can view the interactive project website and final reports here:
👉 **[JSC370 Project Website](https://nkwyk.github.io/JSC370-project/)**

---

## 📂 Repository Structure

The project is organized into two main stages:

* **`midterm/`**: Contains all materials from the Midterm Project phase, including data processing scripts and the initial analysis report.
* **`final_project/`**: This is the core of the repository. It contains the Final Project source files (`.qmd`), data, and the configuration for the Quarto website.

---

## 🚀 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/NKwyk/JSC370-project.git](https://github.com/NKwyk/JSC370-project.git)
    ```

2.  **Data Note**:
    Some raw data files are stored in compressed formats (`.zip` or `.rar`) due to GitHub file size limits. Please extract these files into their respective directories before running the analysis.

3.  **Software Requirements**:
    This project is built using **Quarto**. To render the website locally:
    ```bash
    cd final_project
    quarto render
    ```
