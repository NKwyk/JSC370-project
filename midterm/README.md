# JSC370 Midterm Project - PM2.5 Analysis

This repository contains the midterm analysis for PM2.5 patterns across U.S. metropolitan areas in 2024.

## How to view the report
The full analysis, including interactive visualizations and findings, can be viewed directly by downloading the `JSC370_midterm_PM25_NOAA_AQS2.0.html` file.

## Note on Datasets
Due to GitHub's file size limitations, the original large datasets (.csv files) have been compressed into a ZIP archive (source_data.zip) and uploaded to this repository.
To access the raw data:
Download and extract source_data.zip into the midterm/ directory.
Ensure the file names match those referenced in the .qmd script to run the analysis successfully.
